# Reconnect Directory

Reconnect Directory is a three-page reentry support web app for people returning from incarceration or supervision. It helps users build a trusted circle, save second-chance resources, generate practical plans, draft outreach messages, and log daily check-ins.

## Pages

- `index.html`: dashboard, profile, quick connections, check-ins, and resource matches.
- `directory.html`: searchable resource directory, trusted contact list, saved resources, and local provider entry.
- `programs.html`: AI-style planning tools for reentry goals, outreach messages, and risk interrupts.

## Getting Started

```bash
npm start
```

Open http://localhost:3000 in a browser. The app uses only built-in Node runtime modules, stores demo data in local browser storage, and includes a PWA manifest plus service worker for installable desktop and mobile browser use.

## Verify

npm test
